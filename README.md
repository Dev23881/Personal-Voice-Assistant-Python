# Personal Voice Assistant

A Python-based AI Personal Voice Assistant that performs useful daily tasks using voice output and text commands. The assistant can provide time and date information, search Wikipedia, play YouTube songs, tell jokes, and open commonly used desktop applications.

---

## Features

- Greets the user based on the current time
- Responds with voice output
- Displays current time
- Displays current date
- Searches information from Wikipedia
- Plays songs on YouTube
- Tells random programming jokes
- Opens Google
- Opens YouTube
- Opens GitHub
- Opens Calculator
- Opens Notepad
- Opens File Explorer
- Displays available commands using the Help menu
- Exit command to close the assistant safely

---

## Technologies Used

- Python
- Win32com (SAPI Voice)
- Wikipedia API
- PyWhatKit
- PyJokes
- WebBrowser Module
- OS Module
- Datetime Module

---

## Installation

Install the required libraries using:

```bash
pip install wikipedia pywhatkit pyjokes pywin32
```

---

## How to Run

Run the following command:

```bash
python personal_voice_assistant.py
```

---

## Available Commands

- hello
- how are you
- time
- date
- joke
- who is <person>
- play <song>
- open google
- open youtube
- open github
- open calculator
- open notepad
- open explorer
- help
- exit
